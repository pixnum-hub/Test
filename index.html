Manual

Manual focus, shutter, ISO and white balance use an experimental browser API found mainly on Android Chrome with compatible hardware. Units vary by device; iOS and most desktop browsers won't expose any of these, in which case only Auto is available.

Output

RAW here means uncompressed raw pixel data (.raw), not sensor DNG/CR2 — browsers can't access true camera-sensor RAW. Every pixel is saved exactly as captured, with no compression.
